# Patient Management System

This is a microservices-based patient management system.

## Architecture

The system consists of several microservices, including:
- **API Gateway**: Entry point for client requests.
- **Auth Service**: Handles authentication and authorization.
- **Patient Service**: Manages patient records and related data.
- **Billing Service**: Manages patient billing and payments.
- **Analytics Service**: Provides analytics and reporting capabilities.

## Getting Started

Each service is a Maven project and can be run independently or via Docker if configured.

### Prerequisites
- Java
- Maven
- Any required databases (refer to individual service configurations)
